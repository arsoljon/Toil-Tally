# Toil-Tally
Simple clock-in/clock-out tracker. 
This is an easier method to keep track of how much time I'm spending on each project or tasks. Mainly focused on functionality rather than ui appeal. Which is why I used tkinter for the ui. Implements basic CRUD commands to a SQlite server.
